# 🧠 C/C++ Quiz Application 

An interactive **C/C++ quiz app** built with **React** and **Tailwind CSS**.  
This app allows users to answer multiple-choice questions, track their score, and review results in a clean and responsive UI.  

## ✨ Features

- 🏠 **Start Screen**: Begin the quiz with a single click.  
- ❓ **Quiz Flow**: Answer questions, see progress, and move through the quiz.  
- 📊 **Result Summary**: View score and correctness on completion.  
- 🔄 **Reset Button**: Reset the quiz anytime and start over.  
- 🕒 **Timer**: Track quiz time with a running timer.  
- 📱 **Responsive Design**: Works well on mobile and desktop screens.  

---

## 🛠️ Tech Stack

- ⚛️ **React** – Frontend UI library  
- 🎨 **Tailwind CSS** – Utility-first styling  
- 🔧 **Vite** – Development server and build tool

## 💡 Notes

- The quiz currently includes C/C++ questions first, followed by general programming questions.  
- Question data is stored in `src/App.jsx`, making it easy to add or edit quiz items.  
- The app is a frontend-only React project; there is no backend or database integration yet.
